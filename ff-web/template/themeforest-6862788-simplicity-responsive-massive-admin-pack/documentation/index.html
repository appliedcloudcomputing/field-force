Count To Effect

You have a widget ready with count-to effect enabled here an example of usage:

"<div class="box daily-traffic">

                    <!-- Title Bar Start -->
                    <div class="title-bar">
                      <i class="fa fa-bars"></i>Daily Traffic
                      <div class="close-box">
                        <a href="#"><i class="fa fa-times-circle-o"></i></a>
                      </div>
                    </div>
                    <!-- Title Bar End -->

                    <div class="inner">
                      <div class="col-md-4">
                        <h3>New Visitors</h3>
                        <div class="count-to">739</div>
                      </div>
                      <div class="col-md-4">
                        <h3>Returning Visitors</h3>
                        <div class="count-to">132</div>
                      </div>
                      <div class="col-md-4">
                        <h3>Total Visitors</h3>
                        <div class="count-to">781</div>
                      </div>
                    </div>

                  </div>"
                  
For the elements on which you need the count effect you need to simply add "count-to" and
then you need to add proper JavaScript lib:  "assets/jquery-countto/jquery.count-to.js"

And the JavaScript code:
jQuery('.count-to').each(function(i, item) {
    var end = $(this).html();
    jQuery(item).countTo({from: 0,
      to: end,
      speed: 6000,
      refreshInterval: 50});
});
